# Getting Started with Create Node

## Available Scripts

In the project directory, you can run:

node version : 18

** NOTE
First create .env file 
Add text

host="HOST_NAME"
user="USER_NAME"
password="PASSWORD"
db="DATABASE_NAME"
port=8086

ImagebaseUrl="http://localhost:8086/static/"
**


Step :1
Install Nodemodule
### npm i

Step :2
Create Database 
### npm run db:create

Step :3
Create Table
### npm run table:create


Step :4
Server Start 
### npm run server:start
